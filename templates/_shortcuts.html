Everywhere:
/ or , (slash or comma) - Focus on shortcuts field
ESC - Remove focus from any field
On shortcuts field:
nn - New Note
nb - New Bookmark
nt - New To-Do
n - Go to Notes
p:[num] - Go to page "num"
tag:[query] - Search for notes tagged "query". To search for several tags, you can separate them by commas: "tag1,tag2,tag3"
title:[query] - Search for notes with "query" in the title
kind:[query] - Search for notes with [query] kind. "todo" searches to-dos, "bookmarks" searches bookmarks, "note" searches notes.
[query] - Search for notes with "query" in the title, tag, content or url
On notes list:
x - View note
l - Next page in notes index
h - Previous page in notes index
v - Open note URL (if bookmark)
v - Mark note as done (if To-Do)
On notes list or while viewing note:
j - Next note
k - Previous
e - Edit note in notes index
d - Delete note in notes index
While viewing note:
u - Back to the notes index
r - View raw text
s - View HTML scrape
On note forms:
. (period key) - Focus on Title Field in new/edit Note form
On note forms:
. (period key) - Focus on Title Field in new/edit Note form
